name: Holo-Mind
description: Create and share 3D mind maps with HoloLens.   If you start the App on a second HoloLens, while in the same network with the first one, the app will automatically connect to the first one.  When successfully connected you can sync the mind map using the sync button located in the menu (just look down). From then on both users will edit the same mind map and share the work.  This is a short Playlist, which explains how to connect second HoloLens to a sessions https://www.youtube.com/playlist?list=PLKuCco6d8MNmlFX-XyvG4Pe_ugaX40grN
