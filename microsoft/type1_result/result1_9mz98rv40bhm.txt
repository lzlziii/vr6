name: HoloGauge
description: HoloGauge is a measuring tool to help you to measuring the distance and the area. User only need to place some points on something, the system will use a line to connect points and show the distance between point and point. If the last point and the first point are overlapping, it will become a area and the system will show the area size. If the point is placed on the wrong place, user can use "Redo" to remove the last point, or use "Remove" to clear all points. In addition, The main menu could be move to anywhere as you want.  User could change the display unit. HoloGauge is support Centimeter, Meter, Inch and Feet unit.  Voice commands are available in HoloGauge.  - "Redo" is remove the last point. - "Cut" is create a new line. - "Remove" is clear all points. - "Spatial Enable" / "Spatial Disable" is used to show or hide the spatial map. - "Unit Centimeter" is changed the unit to centimeter. - "Unit Meter" is changed the unit to meter. - "Unit Inch" is changed the unit to inch. - "Unit Feet" is changed the unit to feet.  - "Unit Centimeter" is changed the unit to centimeter.
