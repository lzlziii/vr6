		<h1 class="uk-article-title">
					New Update: NoLimits 2.6 Released			</h1>
	
	
		<p class="uk-article-meta">

		Written on <time datetime="2020-01-20">20 January 2020</time>.
	</p>
	
	
	
			<p>The long-awaited hybrid coaster update has finally arrived. We released the new 2.6 update on the day of the 6th anniversary of NoLimits 2. It is free for all customers of the full version of NoLimits 2.</p>
<div style="padding: 20px; text-align: center;"><a style="padding: 5px;" href="/images/news/update2600/update2600rel_1.jpg" data-uk-lightbox="{group:'nl2600rel'}"><img src="/images/news/update2600/update2600rel_1m.jpg" alt="Screenshot Update 2.6 (1/3)"></a> <a style="padding: 5px;" href="/images/news/update2600/update2600rel_2.jpg" data-uk-lightbox="{group:'nl2600rel'}"><img alt="Screenshot Update 2.6 (2/3)" src="/images/news/update2600/update2600rel_2m.jpg"></a> <a style="padding: 5px;" href="/images/news/update2600/update2600rel_3.jpg" data-uk-lightbox="{group:'nl2600rel'}"><img alt="Screenshot Update 2.6 (3/3)" src="/images/news/update2600/update2600rel_3m.jpg"></a></div>
<p>The new hybrid coaster comes in two variations. One normal version and one LSM launch version with headrests. There is now a total of 43 different coaster styles included in the software. The new wooden support generator was completelly rewritten to fit the new hybrid styles. It gives great results for inline twist inversions out-of-the-box, and only in rare cases manual adjustments are required. There are more new features and optimizations included. Here is a shortened list of the update change log:</p>
<ul>
<li>New coaster style: Hybrid Coaster (standard)</li>
<li>New coaster style: Hybrid Coaster (launched)</li>
<li>Two new parks demonstrating the new coaster styles</li>
<li>Completely rewritten Wooden Support Generator<br data-mce-bogus="1">
<ul>
<li>Automatic supporting of inline twist inversions</li>
<li>Automatic wide supporting for heavily banked tracks</li>
<li>Special patterns for new hybrid coasters</li>
<li>Normal patterns for standard wooden coasters</li>
<li>Generates bolts and nuts</li>
</ul>
</li>
<li>Rectangular steel beams can be without paint</li>
<li>New MSAA anti-aliasing option</li>
<li>Improved Shadows</li>
<li>Improved loading times of parks</li>
</ul>
<p>[The NL Dev Team]</p>
