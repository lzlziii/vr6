          <p><strong>Personal information</strong></p>
          <p>Hyperbolica does not collect any personal identifying information. No accounts are required to use Hyperbolica.</p>
          <p><strong>Analytics</strong></p>
          <p>Hyperbolica collects aggregated, anonymous statistics on which app features are used, hardware specifications, language and country settings, and play session lengths. This data is used to inform development decisions to help improve the experience.</p>
          <p><strong>Crash reporting</strong></p>
          <p>If the Hyperbolica app crashes while running, a crash report may be collected. The crash report contains information about the system where the crash occurred but does not include personal identifying information. Crash reports are used to help identify and resolve software bugs that caused the crash. Crashes in Hyperbolica are reported automatically without prompting the user to submit a crash report.</p>
