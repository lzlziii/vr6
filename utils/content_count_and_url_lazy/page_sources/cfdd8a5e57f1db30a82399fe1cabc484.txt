<p><em>1.3 版 CN (2023 年 4 月 13 日)</em></p>



<p>Synth Riders 是一款虚拟现实 (VR) 节奏游戏，用于打造娱乐体验。本隐私政策描述了 Synth Riders、Kluge Interactive 和 Kluge Strategic, Inc（Synth Riders 的制造商，“我们”、“我们的”和“我们的”）如何使用、共享和保护我们在玩 Synth Riders 游戏时从玩家那里收集。 </p>



<h3 class="wp-block-heading">信息收集</h3>



<ul>
<li>我们收集和存储玩家的唯一平台识别号（玩家 ID）和公开显示名称（玩家显示名称）以及归属于该用户名的分数，并将其存储于由 Pico Interactive 在中国境内托管的数据库中。
</li>



<li>我们收集和存储玩家的头显类型和所使用平台（平台数据）以及玩家在玩游戏时的 IP 地址（网络 ID），并将其存储于由 Pico Interactive 在中国境内托管的数据库中。
</li>



<li>Synth Riders 不会收集或显示玩家的真实姓名、实际地址或联系信息。</li>
</ul>



<h3 class="wp-block-heading">信息使用</h3>



<h4 class="wp-block-heading">玩家 ID、玩家显示名称、平台数据、网络 ID</h4>



<ul>
<li>为玩家提供游戏内排行榜，同时显示他们在平台上好友的详细信息。
</li>



<li>为多人游戏和玩家之间的互动提供便利，包括论坛版块等社交功能。
</li>



<li>支持挑战，使用玩家所选平台的社交功能进行深度链接。
</li>



<li>通过使用速率限制、网络级阻止和访问限制来保护所提供游戏服务的完整性。
</li>
</ul>



<h3 class="wp-block-heading">第三方集成</h3>



<h4 class="wp-block-heading">Pico Interactive – 平台服务</h4>



<p><em>第三方追踪技术：社交功能、活动和移动数据、游戏内购买、游戏设置信息</em></p>



<p>在 Pico 平台上使用 Pico 帐户畅玩 Synth Riders，即可通过该平台自带的社交功能将您在 Synth Riders 内丰富的活动数据共享至 Pico Interactive。其中包括但不限于：关于日期、时间、歌曲和难度的游戏设置数据、记分牌信息、挑战参与详情、Pico 好友、应用程序内购买所需的信息；此外，对于已激活 Pico 体感追踪器的玩家，将收集其在 Pico 平台上的时空虚拟现实追踪数据、健身效果历史和行程。请注意，这些信息并非由 Synth Riders 收集，更多信息请参见 <a class="contact-support" href="https://www.picoxr.com/cn/legal/privacy-policy">Pico Interactive 隐私政策</a>.</p>



<h4 class="wp-block-heading">Pico Interactive – 游戏服务</h4>



<p><em>第三方游戏和多人游戏服务</em></p>



<p>在 Pico 平台上使用 Pico 帐户畅玩 Synth Riders，需要在 Pico Interactive 在中国境内管理的服务器上提供和存储上文“信息收集”部分所述的数据。此信息用于提供多人游戏体验，包括语音聊天 (Photon)、网络排行榜和上文“信息使用”部分所述的论坛版块工具。请注意，这些信息并非由 Synth Riders 收集或存储，更多信息请参见 <a class="contact-support" href="https://www.picoxr.com/cn/legal/privacy-policy">Pico Interactive 隐私政策</a>.</p>
