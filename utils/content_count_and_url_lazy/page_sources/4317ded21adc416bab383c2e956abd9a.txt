			<h2>Sweet Surrender</h2>
			
			<p>We collect statistics for each play session to help us improve the technical performance and gameplay balance. We do not collect any personally identifiable information. We generate a pseudo-unique id for each player to help us identify the number of players but we are technically unable to tie this to any player. This id is displayed to the player in the settings menu of the game. When contacting us with a support or privacy request, you are welcome (but never obligated) to reference this id so that we can track down further details about your account.</p>

			<p>As of September 12th 2021 this feature is disabled. We're leaving the note above for historical purposes and if ever turn this on again.</p>
