																			<h1>为何要采用抢先体验这种模式？</h1>
										“For cyubeVR, the "Early Access" label is a positive label to immediately show that the game is in active development. It is supposed to be the promise that you can expect a <strong>living game with constant free updates being released</strong> that further improve the game, and not a dead game that will never change, or a game that requires you to buy new DLCs to get any new content.<br><br>As of January 2023, cyubeVR has been in full-time development for exactly 7 years. That's a long time! The game is generally considered super polished by now - so it might be very different than most other "Early Access" games in that regard. You should <strong>not</strong> expect the game to be buggy or unpolished in any way. One of the highest priorities is ensuring that <strong>there are 0 known bugs in the game</strong>. Stability, good performance, and being bug-free are most important at any time during development. cyubeVR should always feel super polished. If a bug is reported, it will be fixed before any new feature is added. <br><br>Receiving a lot of feedback from the players who play and love the game at any point in time is great to ensure the game always ideally fulfills the expectations of most players. The main place for feedback and general community discussion is the official <a href="https://steamcommunity.com/linkfilter/?url=http://discord.gg/cyubeVR" target="_blank" rel=" noopener">cyubeVR Discord server</a> where we have an amazing community of over 3800 people already. And if you join us there, it will surely be an even more amazing community!<br><br><a href="https://steamcommunity.com/linkfilter/?url=http://roadmap.cyubevr.com" target="_blank" rel=" noopener">If you want to see what's planned for the game in future updates, you can see the official roadmap (Trello) for the game here.</a> The two most requested features that are planned to be added are survival mechanics and multiplayer, which will significantly expand the target audience of the game to more players over time until cyubeVR hopefully eventually becomes the most popular VR game that exists :)”

										<h1>这款游戏的抢先体验状态大约持续多久？</h1>
										“cyubeVR is a game that can grow forever, and the developer intends to work on it for a long time to come - logically, when there are constant updates, the game can always be expected to be better two months later than two months earlier. So at any current point in time, it's "early" compared to what it will be "later", and that's good! <br><br>If everything goes well, cyubeVR is a game that will hopefully never be considered "finished" or "complete" because there will always be more content added to the game, at least as long as the developer is able to afford to work on the game, which will hopefully be forever (If you want to help with that more than just through buying the game on Steam, you can also consider to <a href="https://steamcommunity.com/linkfilter/?url=https://www.patreon.com/stonebrickstudios" target="_blank" rel=" noopener">support the development of cyubeVR on Patreon</a>).<br><br>So if it is possible, there will still be free updates for cyubeVR in 5 and 10 years from now, and that then of course means playing the version of cyubeVR that exists in 5 years will be an "Early" version compared to the version that exists in 10 years. So the game will ideally stay this definition of "Early Access" forever!”

										<h1>计划中的完整版本和抢先体验版本到底有多少不同？</h1>
										“Currently cyubeVR is singleplayer, and one of the most requested features is multiplayer. Multiplayer is definitely planned for the future! Of course you will still be able to play this in singleplayer, just like now.<br><br>Also, the game is currently a very chill and peaceful sandbox game primarily about exploration, crafting, and most importantly, building, without any survival aspect apart from fall damage. It is a popular request to add more survival-related gameplay mechanics, like melee combat, so that's also planned. It will stay optional though, so anyone who likes the current peaceful atmosphere of the game will definitely be able to keep playing that way and focus on the creative aspect of the game.<br><br><a href="https://steamcommunity.com/linkfilter/?url=http://roadmap.cyubevr.com" target="_blank" rel=" noopener">And same as above, let's link the official roadmap here again!</a> You can of course expect many more changes to the game in the next few years and (hopefully) decades. With a long-term project like cyubeVR, no one can even know yet what the most popular requests from the community will look like in 10 years from now.”

										<h1>抢先体验版本的现状如何？</h1>

And the target audience and scope of the game will continue to grow over time.<br><br>It is important that emphasize again that you should <strong>not</strong> expect the game to be buggy or unpolished in any way. New features or content is always tested in beta builds by beta testers in the <a href="https://steamcommunity.com/linkfilter/?url=http://discord.gg/cyubeVR" target="_blank" rel=" noopener">cyubeVR Discord server</a>, so by the time anything new is released in an update in the regular stable build of the game, it should all be guaranteed to work well and feel very polished.”
