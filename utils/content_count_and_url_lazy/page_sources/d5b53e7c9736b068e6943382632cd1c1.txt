<h2 class="fusion-responsive-typography-calculated" style="--fontsize: 40; line-height: 1.2; --fontSize:40;" data-fontsize="40" data-lineheight="48px">YOUR DATA IN OUR GAMES</h2>
<p>Shotgun Sports utilizes online leaderboards, meaning that player’s in game data (scores and usernames) is sent to the Oculus platform and in some cases it shown online, e.g. on the Shotgun Sports gameplay video.<br><span style="background-color: rgba(255, 255, 255, 0); color: var(--body_typography-color); font-family: var(--body_typography-font-family); font-size: var(--body_typography-font-size); font-style: var(--body_typography-font-style,normal); font-weight: var(--body_typography-font-weight); letter-spacing: var(--body_typography-letter-spacing);">To allow this, our games has to track current users identification names or numbers, and their best current score.<br></span><span style="background-color: rgba(255, 255, 255, 0); color: var(--body_typography-color); font-family: var(--body_typography-font-family); font-size: var(--body_typography-font-size); font-style: var(--body_typography-font-style,normal); font-weight: var(--body_typography-font-weight); letter-spacing: var(--body_typography-letter-spacing);">It possible that we might change the game related data that we analyze or store. Any modification is made to make the game more fun to play.<br></span><span style="background-color: rgba(255, 255, 255, 0); color: var(--body_typography-color); font-family: var(--body_typography-font-family); font-size: var(--body_typography-font-size); font-style: var(--body_typography-font-style,normal); font-weight: var(--body_typography-font-weight); letter-spacing: var(--body_typography-letter-spacing);">Other than the above mentioned cases, we do not store any player’ specific data.<br></span><span style="background-color: rgba(255, 255, 255, 0); color: var(--body_typography-color); font-family: var(--body_typography-font-family); font-size: var(--body_typography-font-size); font-style: var(--body_typography-font-style,normal); font-weight: var(--body_typography-font-weight); letter-spacing: var(--body_typography-letter-spacing);">Shotgun Sports built using the Unity engine, therefore we do not use any crash reporting or analytics offered by Unity.</span></p>
