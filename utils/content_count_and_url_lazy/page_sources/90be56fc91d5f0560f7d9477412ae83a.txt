									<p>通过 Windows Mixed Reality for SteamVR，用户可以在 Windows Mixed Reality 沉浸式头戴显示设备上运行 SteamVR 体验。 安装 Windows Mixed Reality for SteamVR 后，用户可以从他们的桌面或 Steam 库启动喜爱的 SteamVR 应用程序，并直接在他们的 Windows 头戴显示设备上玩。</p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#get-your-pc-ready" aria-label="节标题：准备好电脑"></a><h2 id="get-your-pc-ready" class="heading-anchor">准备好电脑</h2></div>
<ul>
<li>确保没有挂起的更新：选择<strong>“开始&gt;设置”“&gt;更新&amp;安全&gt;Windows 更新</strong>”。 如果有可用更新，请选择“立即安装”。 如果没有可用更新，请选择“检查是否有更新”，然后安装任何新更新。</li>
<li>Steam 上的应用和内容对电脑的要求各不相同。 请参阅每个游戏的最低要求。 配备 GTX 1070 显卡（或同等显卡）和 Intel® Core™ i7 处理器的电脑应该能为众多游戏提供良好的体验。</li>
<li>如果尚未设置 <a href="set-up-windows-mixed-reality" data-linktype="relative-path">Windows Mixed Reality</a>，请进行设置。</li>
</ul>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#set-up-windows-mixed-reality-for-steamvr" aria-label="节标题：设置 Windows Mixed Reality for SteamVR"></a><h2 id="set-up-windows-mixed-reality-for-steamvr" class="heading-anchor">设置 Windows Mixed Reality for SteamVR</h2></div>
<ol>
<li><a href="https://steamcdn-a.akamaihd.net/client/installer/SteamWindowsMRInstaller.exe" data-linktype="external">下载并安装 SteamVR。</a></li>
<li>准备就绪后，启动 SteamVR。 SteamVR 教程应会自动启动。</li>
</ol>
<blockquote>
<p>              “注意：”对于 SteamVR 设置的高级故障排除，请确保已安装以下软件组件：</p>
<ol>
<li>安装 <a href="http://store.steampowered.com/about/" data-linktype="external">Steam</a> 并“登录”或“创建一个新帐户”。</li>
<li>安装 <a href="https://store.steampowered.com/app/250820/SteamVR/" data-linktype="external">SteamVR</a>。 插入头戴显示设备后，启动 Steam，应会看到一个对话框，提示你安装 SteamVR。 请按照对话框中的提示进行安装。
* 如果未看到弹出窗口，请导航到“库”的“工具”部分来安装 SteamVR。 在列表中找到 SteamVR，然后右键单击并选择“安装游戏”。</li>
<li>安装 <a href="https://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/" data-linktype="external">Windows Mixed Reality for SteamVR</a>。</li>
</ol>
</blockquote>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#set-up-windows-mixed-reality-for-steamvr-in-an-environment-without-internet-access" aria-label="节标题：在一个无法访问 Internet 的环境中设置 Windows Mixed Reality for SteamVR"></a><h2 id="set-up-windows-mixed-reality-for-steamvr-in-an-environment-without-internet-access" class="heading-anchor">在一个无法访问 Internet 的环境中设置 Windows Mixed Reality for SteamVR</h2></div>
<p>              将必要的介质存储在便携式存储设备上</p>
<ol>
<li>在可完全访问 Internet 的电脑上使用 <a href="http://store.steampowered.com/about/" data-linktype="external">Steam</a> 按照上述说明安装 <a href="https://store.steampowered.com/app/250820/SteamVR/" data-linktype="external">SteamVR</a> 和 <a href="https://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/" data-linktype="external">Windows Mixed Reality for SteamVR</a>。</li>
<li>在 Steam 中，打开“库”部分并找到标记为“工具”的部分。</li>
<li>安装 SteamVR 后，右键单击“SteamVR”条目，然后在出现的弹出菜单中单击“属性”条目。</li>
<li>包含多个选项卡的新窗口随即打开。 选择“本地文件”选项卡并单击标记为“浏览本地文件”的按钮。</li>
<li>包含 SteamVR 运行时的目录随即打开。 将整个目录（名为 SteamVR）复制到所选的便携式介质（例如 USB 拇指驱动器）上。</li>
<li>对 Windows Mixed Reality for SteamVR 以及你希望在目标电脑上安装的任何与 SteamVR 兼容的应用执行相同的操作。</li>
</ol>

<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#play-steamvr-games" aria-label="节标题：玩 SteamVR 游戏"></a><h2 id="play-steamvr-games" class="heading-anchor">玩 SteamVR 游戏</h2></div>
<ol>
<li>将头戴显示设备连接到电脑并打开运动控制器。</li>
<li>加载 Windows Mixed Reality 主页并显示控制器后，打开桌面上的 Steam 应用。</li>
<li>使用 Steam 应用从 Steam 库中启动 SteamVR 游戏。</li>
</ol>
<p>              “提示”：若要在不摘下头戴显示设备的情况下启动 SteamVR 游戏，请使用桌面应用（“启动”&gt;“桌面”）在 Windows Mixed Reality 中查看电脑桌面并与之交互。</p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#using-motion-controllers-with-steamvr" aria-label="节标题：将运动控制器与 SteamVR 结合使用"></a><h2 id="using-motion-controllers-with-steamvr" class="heading-anchor">将运动控制器与 SteamVR 结合使用</h2></div>
<p>你将在不同游戏中以不同方式使用运动控制器。 以下是一些可帮助你入门的基础知识：</p>
<ul>
<li>若要打开 Steam 仪表板，请直接按下左或右控制杆。</li>
<li>若要退出 SteamVR 游戏并返回 Windows Mixed Reality 主页，请按 Windows 按钮。</li>
</ul>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#changing-the-resolution" aria-label="节标题：更改分辨率"></a><h2 id="changing-the-resolution" class="heading-anchor">更改分辨率</h2></div>
<p>如果想以更高的分辨率玩游戏，可以随时在“SteamVR”-&gt;设置”-&gt;“应用程序”窗口中调整应用程序分辨率滑块。 **使用分辨率更高的乘数时，可以预期到游戏会对电脑造成更大负担。 如果增加乘数并看到性能下降，请将滑块重新调整为默认级别并重启游戏，以确保更改生效。<img src="images/steamvr_settings_applications.png" alt="调整应用程序分辨率" data-linktype="relative-path"></p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#using-multiple-headsets" aria-label="节标题：使用多个头戴显示设备"></a><h2 id="using-multiple-headsets" class="heading-anchor">使用多个头戴显示设备</h2></div>
<p>如果你是一名 VR 爱好者，你可能经常在同一台电脑上使用多个 VR 头戴显示设备。 如果是这种情况，请注意，只要插入了 Windows Mixed Reality 头戴显示设备，SteamVR 游戏将始终在该设备上启动。 如果想在另一个头戴显示设备上启动 SteamVR 游戏，请确保在继续之前先拔下 Windows Mixed Reality 头戴显示设备。</p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#preview-programs" aria-label="节标题：预览程序"></a><h2 id="preview-programs" class="heading-anchor">预览程序</h2></div>
<p>我们会定期发布更新，以提高在 Windows Mixed Reality 沉浸式头戴显示设备上使用 SteamVR 的性能、可靠性和整体体验。 虽然这些预览程序都不是必需的，但如果你想更快更频繁地获得更新（并就这些更新向我们提供反馈！），我们鼓励你使用它们。</p>
<div class="heading-wrapper" data-heading-level="h3"><a class="anchor-link docon docon-link" href="#windows-mixed-reality-for-steamvr-beta" aria-label="节标题：Windows Mixed Reality for SteamVR Beta 版"></a><h3 id="windows-mixed-reality-for-steamvr-beta" class="heading-anchor">Windows Mixed Reality for SteamVR Beta 版</h3></div>
<p>Windows Mixed Reality for SteamVR 是你从 Steam 商店安装的组件，通过该组件可将 SteamVR 与 Windows Mixed Reality 头戴显示设备配合使用。  我们会定期发布该“桥”的更新，并且 Steam 会自动安装它们。</p>
<p>如果想更频繁地获得更新，建议你加入我们的公开 Beta 版。  更新首先会面向 Beta 版受众，我们会利用他们的反馈来确保更新质量过关，然后再向所有用户发布。  如果你未加入 Beta 版计划，你最终也会获得同样的修补程序和功能，只是会在 Beta 版用户对其进行测试之后。</p>
<p>若要加入，请执行以下操作：</p>
<ol>
<li>在 Steam 中，使用“库”菜单下的下拉菜单筛选“软件”。</li>
<li>在列表中，右键单击“Windows Mixed Reality for SteamVR”并选择“属性”。</li>
<li>选择“Beta”选项卡。</li>
<li>选择加入“Beta 版 - 公开 Beta 版”，然后选择“关闭”进行确认。 Beta 访问代码字段应留空。</li>
</ol>
<div class="heading-wrapper" data-heading-level="h3"><a class="anchor-link docon docon-link" href="#steamvr-beta" aria-label="节标题：SteamVR Beta 版"></a><h3 id="steamvr-beta" class="heading-anchor">SteamVR Beta 版</h3></div>
<p>SteamVR 由 Valve 构建和发布，适用于所有 SteamVR 头戴显示设备。  它遵循一种类似的模式，即先向 Beta 版成员发布更新，然后再发布给所有用户。</p>
<p>若要加入，请执行以下操作：</p>
<ol>
<li>在 Steam 中，使用“库”菜单下的下拉菜单筛选“工具”。</li>
<li>在列表中，右键单击“SteamVR”并选择“属性”。</li>
<li>选择“Beta”选项卡。</li>
<li>选择加入“Beta 版 - 公开 Beta 版”，然后选择“关闭”进行确认。  beta 访问代码字段应留空。<img src="images/steamvr-beta.png" alt="在 SteamVR 的属性对话框中切换到 SteamVR beta 版本" data-linktype="relative-path"></li>
</ol>
<div class="heading-wrapper" data-heading-level="h3"><a class="anchor-link docon docon-link" href="#windows-insider-program" aria-label="节标题：Windows 预览体验计划"></a><h3 id="windows-insider-program" class="heading-anchor">Windows 预览体验计划</h3></div>
<p>Windows Mixed Reality 是 Windows 10 和 Windows 11 的一部分。  Windows OS 附带了许多影响 SteamVR 用户的修复程序和功能。  若要试用最新的 Windows 10 和 Windows 11 预览版，建议加入 <a href="https://insider.windows.com" data-linktype="external">Windows 预览体验计划</a>。</p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#enabling-motion-reprojection-for-steamvr-apps" aria-label="节标题：为 SteamVR 应用启用运动重投影"></a><h2 id="enabling-motion-reprojection-for-steamvr-apps" class="heading-anchor">为 SteamVR 应用启用运动重投影</h2></div>
<p>Windows Mixed Reality for SteamVR 具有一项实验性的运动重投影功能，使 90 FPS 重投影更加流畅。</p>
<p>启用运动重投影后，所有 Steam VR 游戏在名义上都将以 ½ 帧速率（45 fps 而不是 90 FPS）呈现，而 Windows Mixed Reality for SteamVR 使用 GPU 生成的运动矢量来推断下一帧。 对于在给定电脑上可靠地达到 60 FPS+ 的 SteamVR 游戏而言，这应该会带来稳定的 90 FPS 体验，偶尔出现伪影，同时保持舒适的体验。</p>
<p>可用的运动重投影模式如下所示：</p>
<ul>
<li>              “SteamVR 每应用设置”：允许你通过 SteamVR 设置 UI 控制运动重投影。 然后，可以打开 SteamVR 设置，转至“视频”&gt;“每应用程序视频设置”，并选择“运动平滑处理”选项。</li>
<li>              “自动”：如果游戏呈现速度过慢，无法保持 90 FPS，请将运动重投影设置为自动打开。 当游戏开始保持 90 FPS 或开始以低于 45 FPS 呈现时，运动重投影将关闭。 异步旋转重投影始终处于启用状态。</li>
<li>              “运动向量”：强制应用程序始终以运动矢量重投影的半帧率运行。</li>
<li>              “无”：禁用运动重投影。</li>
</ul>
<p>              “预期视觉假象”</p>
<ol>
<li>使用大于 150% 的应用程序分辨率时，可能会出现模糊。 使用运动重投影时，建议使用小于 150% 的值。</li>
<li>鲜明对比的边缘或文本，尤其是在游戏中的 HUD 或菜单上，可能会因去除遮挡而暂时变形或扭曲。</li>
<li>SteamVR Home 和电脑上无法可靠地达到 50-60 FPS 的其他许多游戏在此模式下的体验仍然很差。</li>
<li>据报道，某些游戏以 50% 的速度运行或延迟增加（滞后）。 通过下面的 <a href="filing-feedback" data-linktype="relative-path">Windows 反馈中心</a>说明报告这些游戏。</li>
</ol>
<p>最初，我们为最新一代 NVidia GPU 提供实验性支持。 我们将继续迭代和改进对更多 GPU 的运动重投影支持，我们很期待收到你的反馈。</p>
<p>              “支持的 GPU：”Nvidia GeForce GTX1060、AMD RX470 或更高版本（安装了兼容 Windows Mixed Reality 的图形驱动程序）。</p>
<p>启用运动重投影：</p>
<ol>
<li>确保已按照上述说明选择加入 “Windows Mixed Reality for SteamVR Beta 版”。</li>
<li>打开 SteamVR 仪表板。</li>
<li>选择左侧带有 Windows Mixed Reality 徽标的按钮以打开 “Windows Mixed Reality for SteamVR” 设置。</li>
<li>在弹出的 UI 中，选择“图形”选项卡。</li>
<li>为“默认 SteamVR 应用运动重投影模式”选择“自动”以启用自动运动重投影。</li>
</ol>
<p><img src="images/settingsux-enable-lsr.gif" alt="使用设置启用 LSR &amp; LSR 指示器UX" data-linktype="relative-path"></p>
<p>              “运动重投影指示器”</p>
<p>运动重投影指示器有助于诊断实验性自动运动重投影功能的问题。 当设置为 true 时，你将在自动运动重投影期间在头戴显示设备的左上方看到一个指示器。 该指示器的颜色和位置对应于当前的运动重投影模式 - 有关示例，请参阅下图。</p>
<p><img src="images/mvlsrindicator.png" alt="mvLSR 指示器" data-linktype="relative-path"></p>
<p>绿色 = 运动重投影处于关闭状态，因为应用程序能够以全帧速率呈现。</p>
<p>青色 = 运动重投影处于打开状态，因为应用程序受 CPU 限制。</p>
<p>蓝色 = 运动重投影处于打开状态，因为应用程序受 GPU 限制。</p>
<p>红色 = 运动重投影处于关闭状态，因为应用程序正在以低于一半的帧速率运行；如果已启用，请尝试减少超级采样。</p>
<p>绿色 + 青色 + 蓝色 = 运动重投影处于半帧率模式或应用程序请求运动重投影。</p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#sharing-feedback-on-steamvr" aria-label="节标题：共享有关 SteamVR 的反馈"></a><h2 id="sharing-feedback-on-steamvr" class="heading-anchor">共享有关 SteamVR 的反馈</h2></div>
<p>在改进 Windows Mixed Reality SteamVR 体验方面，你的反馈非常宝贵。 通过 <a href="filing-feedback" data-linktype="relative-path">Windows 反馈中心</a>提交所有反馈和错误。 请遵循以下建议，以帮助我们充分利用你的反馈：</p>
<ol>
<li>在反馈中心，在顶部的“这是什么类型的反馈？”部分指出你正在报告新问题。</li>
<li>选择“混合现实”类别和“应用”子类别。</li>
<li>在问题摘要中注明“SteamVR”一词。 这有助于我们找到你的反馈。</li>
<li>描述你遇到问题时使用的 SteamVR 游戏或应用程序。</li>
<li>考虑在你的反馈中附上一份 SteamVR 系统报告。 这将提供更多日志，帮助我们诊断你的问题。
<ol>
<li>在 SteamVR 窗口（显示控制器状态的小窗口）上，选择游戏以打开菜单。</li>
<li>选择“创建系统报告”。</li>
<li>保存到文件。</li>
<li>将生成的文件直接附加到“反馈中心”条目。</li>
</ol>
</li>
<li>如果你的反馈与 SteamVR 性能有关，请收集混合现实性能跟踪： <ol>
<li>选择“重新创建我的问题”按钮。</li>
<li>在“包含的数据类型”旁边的下拉菜单中，选择“混合现实性能”。</li>
<li>确保游戏正在运行并选择“开始捕获”。</li>
<li>花几秒钟玩游戏来捕获跟踪内容。 捕获跟踪的时间不要超过 10-15 秒，否则会太大而无法提交。</li>
<li>选择“停止捕获”。</li>
</ol>
</li>
<li>完成其余字段后，选择“提交”。</li>
</ol>
<p>如果有任何问题或意见想要分享，也可以通过我们的 <a href="http://steamcommunity.com/app/719950/discussions/" data-linktype="external">Steam 论坛</a>与我们联系。</p>
