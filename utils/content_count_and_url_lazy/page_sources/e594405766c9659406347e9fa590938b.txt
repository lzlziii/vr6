<li>Design Incredible Immersive Interfaces, Spaces, &amp; Experiences.</li>
<li>Bring Maquette Content Into Unity With the Unity Add-on.</li>
</ul>

<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#export-content-from-maquette" aria-label="Section titled: Export content from Maquette"></a><h2 id="export-content-from-maquette" class="heading-anchor">Export content from Maquette</h2></div>
<div class="heading-wrapper" data-heading-level="h3"><a class="anchor-link docon docon-link" href="#part-1---export-content-from-maquette" aria-label="Section titled: Part 1 - Export content from Maquette"></a><h3 id="part-1---export-content-from-maquette" class="heading-anchor">Part 1 - Export content from Maquette</h3></div>
<ol>
<li><p>Use the Select Tool to highlight what you want to export to Unity.</p>
</li>
<li><p>Highlight your desired content, and select the <strong>Export</strong> icon at the bottom of the Select menu on your hand.</p>
</li>
<li><p>In the context menu that appears, rename the content for export, and select Export to either .FBX, .GLB, or .GLTF file types.</p>
</li>
<li><p>A File Explorer window will appear on your desktop showing your exported files, and you can always find Maquette's exported files in <strong>~Documents/Maquette/Exports/</strong></p>
</li>
</ol>
<div class="heading-wrapper" data-heading-level="h3"><a class="anchor-link docon docon-link" href="#part-2---import-the-maquette-unity-add-on-into-unity" aria-label="Section titled: Part 2 - Import the Maquette Unity Add-On into Unity"></a><h3 id="part-2---import-the-maquette-unity-add-on-into-unity" class="heading-anchor">Part 2 - Import the Maquette Unity Add-On into Unity</h3></div>
<p>The Maquette Unity Editor Add-On package ensures .FBX and .GLTF files exported from Maquette appear in Unity in the same way they appear in Maquette. Without importing this package, Unity will not display Maquette's materials and images.</p>
<ol>
<li><p>Open the Unity project where you intend to use your Maquette content within.</p>
</li>
<li><p>In the Unity taskbar, select <strong>Assets &gt; Import Package &gt; Custom Package</strong>, and open the Microsoft Maquette Unity Editor Add-On package, downloaded from the button above.</p>
</li>
<li><p>Select <strong>Import</strong> at the bottom right of the window that appears.</p>
</li>
</ol>
<div class="nextstepaction">
<p><a href="https://aka.ms/MaquetteDownloadAddOn" data-linktype="external" class="button button-primary button-filled has-text-wrap">Download the Unity Add-on</a></p>
</div>
<div class="heading-wrapper" data-heading-level="h3"><a class="anchor-link docon docon-link" href="#part-3---import-maquette-content-into-unity-with-the-unity-add-on" aria-label="Section titled: Part 3 - Import Maquette Content into Unity with the Unity Add-On"></a><h3 id="part-3---import-maquette-content-into-unity-with-the-unity-add-on" class="heading-anchor">Part 3 - Import Maquette Content into Unity with the Unity Add-On</h3></div>
<ol>
<li><p>Import the Maquette Unity Editor Add-On package (See the section below for more info) into your Unity project.</p>
</li>
<li><p>Bring your .FBX, .GLB, or .GLTF files exported from Maquette into your Unity project. Unity will preserve the grouping hierarchy of your content, and the materials and images as they appear in Maquette, with your imported the Microsoft Maquette Unity Editor Add-On.</p>
</li>
</ol>
