                                                $J('#highlight_player_area').on("touchmove", function (event) {

                                                        return;

                                                    var ydiff = Math.abs( g_TouchEventStart.originalEvent.changedTouches[0].screenY - event.originalEvent.changedTouches[0].screenY );

                                                    // To reduce unintended page scroll during a horizontal swipe block the default handler when it looks like the user
                                                    // is in the process of performaing a horizontal scroll
                                                    if ( xdiff > ydiff && ( typeof event.cancelable !== 'boolean' || event.cancelable ) )
                                                        event.preventDefault();

                                                });
                                                $J('#highlight_player_area').on("touchend", function (eventEnd) {

                                                    if ( typeof g_TouchEventStart.originalEvent.changedTouches[0] == undefined ||
                                                        typeof eventEnd.originalEvent.changedTouches[0] == undefined )
                                                        return;

                                                    var xdiff = g_TouchEventStart.originalEvent.changedTouches[0].screenX - eventEnd.originalEvent.changedTouches[0].screenX;
                                                    var ydiff = g_TouchEventStart.originalEvent.changedTouches[0].screenY - eventEnd.originalEvent.changedTouches[0].screenY;

                                                    // clear out the touch start event
                                                    g_TouchEventStart = null;

                                                    // ignore the swipe if it was short and could've been intended as a tap, or the
                                                    // direction was more vertical than horizontal
                                                    if ( Math.abs( xdiff ) < 20 || Math.abs( ydiff ) > Math.abs( xdiff ) )
                                                        return;

                                                    if ( typeof event.cancelable !== 'boolean' || event.cancelable )
                                                        event.preventDefault();

                                                    // move forward or backward but don't wrap as that would be a jarring user experience
                                                    if ( xdiff > 0 )

                                                        if ( $NextItem.length )

					终于全成就了。<br>这里给大家一点游玩建议吧：<br>1、游戏初期给的副武器防空炮可以说是游戏里反舰伤害最高的武器了，但是注意对护盾伤害约等于没有，建议用机动飞到对方护盾内部以后直接对着舰体喷，基本上一发可以打爆护舰艇，多来几下就能报销驱逐舰和护卫舰的舰桥，在进行对舰任务的时候一定要考虑带上它。<br>2、就像之前提到的，这游戏里的大船是存在部位破坏的，靠的足够近以后iff就会给你标出敌舰的舰桥、引擎、炮塔等部件，打爆舰桥以后会直接让报销对手的护盾以及一大半血条，而击破炮塔和引擎以后会让它们停机一定的时间，但往往可以给我方的大船争取到更多的开火机会以及生存几率，尤其在前期你的护卫舰只有伤害高但是开火距离不忍直视的光矛，给它争取到最佳的开火时机会比你独自输出（尤其是还没有解锁航弹的前期）重要许多。<br>3、你的僚机有时会比你更恐怖。在之后的战斗中你会在队伍中再增添两架拦截机，它们给了你失败后反败为胜的筹码，而哪怕你坠毁了，再击落三架敌方单位（你僚机击落的也算）包括敌方的集装箱，你就可以复活一架自机，所以如果你技术得当可以在战斗中续航很久。而且你的僚机在狗斗上可能比你强上很多，一些我要花很长时间才能独自解决的ace机结果交给ai拦截机你可能一个不留神战斗就结束了（但如果被鱼雷盯上就一个个都变老六了），如果在高难度下这个技巧会非常有用，因为在最高难度下旗舰出现时还会伴随一种跃迁干扰舰，这会极大的增加跑路的难度，届时你将不得不顶着巡洋舰的重炮再去把干扰舰炸掉。<br>4、敌人打出来的鱼雷你是可以拦截的！！！尤其是在空旷的环境下，你搁那里机动半天可能都不如刹车然后枪点鱼雷来的高效，而且你也可以给僚机拦截掉朝它们打来的鱼雷，大部分拦截机在面对鱼雷时都是老六枪法，而如果你不给大船装点防御而是装的破片炮的话，那也记得留意一下它们的血量。当然，消灭鱼雷的最好方式永远是拆掉敌方的鱼雷发射井，或者干脆给它来一发航弹。<br>5、关于狩猎旗舰，这里推荐大家通过一次主线以后再去挑战。每一关的旗舰配置实际上都不一样，但打法上大同小异，无外乎就是炸掉对方的舰桥，然后直接航弹伺候，此时要注意对方有没有维修无人机（repair drone），只要它们出现就第一时间干掉，毕竟它们的数量有限而且只要机炮扫个不到两秒就能收走,维修无人机的图标是一个小的宽型十字，和航母以及驱逐舰的标志非常想，切记要先在战略视图里确认对方的具体配置如何。<br>6、记住这个游戏是有战术模式的，大部分操作都和家园系列一样，按m键才能移动。你的主力舰在你不下达指令的情况下很可能会OMO，尤其是旗舰和敌方增援进场它们相隔甚远的时候。<br><br>     这里再推荐几套狩猎旗舰的常用配置（重型武器默认带航弹）：<br>1、 装备：护盾+超空间跳跃（90点的那个），武器带机炮和霰弹（初始的防空炮），让己方战舰先上然后突脸，如果有无人机优先点掉无人机，随后突入护盾内丢航弹（航弹会随着你的机体的惯性移动)然后火力全开炸舰桥。<br>2、装备：护盾（强烈推荐护盾）+神风装甲，武器随意，能让你撑到旗舰出现就行。战斗时直接让两架拦截机一起自杀式冲锋就行，可能会比较考验你的机动水平，一般只要两架满载的神风机就可以送对面上天了，若是不行就苟住然后尽力等击杀三架以后的出现的补给机。<br>       战术上基本都是一波流，但这往往比稳步推进要高效特别多，巡洋舰的火力往往会让我方僚机和主力舰伤亡惨重，<span class="bb_spoiler"><span> 而且作为“龙” 实际上在剧情上你是死不掉的</span></span>，成就必须要在Harbringer难度下才能解锁，在这种难度下往往战术上的优势会更明显，况且游戏给出的各类装备实际上都很鼓励自毁233333。<br><br>游戏的剧情实际上很简单，一个银河帝国的帝王驾崩，他麾下的近卫军团“龙”开始了对叛军无休止的复仇，<span class="bb_spoiler"><span>这场复仇一直持续到“龙”摧毁了所有的叛军以及帝国的首都，没错，他老人家自己的首都</span></span>。<br>大部分的剧情都可以在codex里看到，而当游戏通关过一次以后也都会解锁，游戏的结局需要最高难度通关一次就可以全部解锁，慢慢打就可以。					<div class="gradient"></div>
