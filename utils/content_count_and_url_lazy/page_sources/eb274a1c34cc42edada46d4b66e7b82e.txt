These are one-pixel image files, linked to the server of Inxmail, thus making it possible to evaluate user behavior.&nbsp; This takes place by the collecting of server-log files, see 1, like the time of calling up the e-mail and the corresponding IP address as well as the web beacons assigned to your e-mail address and linked with a personal ID. A user profile is created with the data obtained, in order to tailor the newsletter to your interests. You can object to such tracking at any time by unsubscribing the newsletter.</p></div>
