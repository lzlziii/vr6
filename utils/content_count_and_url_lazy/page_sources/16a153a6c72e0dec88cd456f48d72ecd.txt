                                                    // To reduce unintended page scroll during a horizontal swipe block the default handler when it looks like the user
                                                    // is in the process of performaing a horizontal scroll

                                                        event.preventDefault();

                                                        event.preventDefault();

                                                    if ( xdiff > 0 )

				<div class="content">
					自《幻》事件之后，大多数玩家对于国产游戏的不认可似乎已经到了无法挽救的零界点。不过，这样似乎并没有产生太大的影响，国产单机游戏已无影响可言，国产VR游戏亦从未有过影响。<br><br>这款国产VR游戏，算是比较特殊一点，因为我从中看到了不少创新点。<br><br>——————————————————————————————<br><br>从把游戏加入愿望单到打折入手游戏，我都一直看中一点，就是游戏中对于玩家身体的模拟。就是虚拟的身体会跟着人手部的移动来产生连贯反应，据说叫什么全身IK自动演算，反正就是一种技术。<br><br>它可以让人感觉到自己的虚拟身体十分的真实。从我的实际体验来看，虽然还达不到十分真实的效果吧，但这一点确实要比其他VR游戏做得要好不少。我还是相当满意的。扭来扭去就是很好玩啊。<br><br>游戏画质也相当不错，咋看之下还以为用的是虚幻引擎，但是实际上是用了Unity。可以看得出这个团队的美术功底还是很深厚的。而且游戏的物理效果还是值得点赞的，各种击退啊，挥砍体验也看起来很真实，同时打击感可以说是比较良好的。<br><br>游戏玩法上，目前采用了突突突射击+砍砍砍的路子+无限子弹时间的路子，这三种元素的搭配算是中规中矩。整体来看，游戏是适合所有喜欢VR动作射击游戏的玩家。<br><br>经过更新之后，游戏的内容已经开始变多，看游戏公告，每一次更新的内容都不少，又是增加又是修复的，希望他们继续努力吧。<br><br>说一个比较特别的地方，就是游戏里面的武士刀无限切割效果相当浮夸。平时玩其他VR游戏，挥刀砍过去敌人都是死掉或者爆炸，但是这个游戏里面切过去是一刀两段，速度够快竟然可以达到大卸八块的效果，哈哈。<br><br>游戏的不足之处呢，就是还有一些BUG需要处理，该优化的地方，还是尽快优化。另外，AI可以弄得再聪明一点，这样调整性更高。随着游戏内容的进一步丰富，希望可以加入存档点的设计，不然就很尴尬。<br><br>还有一点，游戏一开始其实可以以抢先体验和更加低廉的价格来上线的。但是，这两项都没有做到，难免就会被原价买入的玩家抱怨游戏时长不够。<br><br>总体来讲，站在我自己的角度，我还是很愿意把这款游戏推荐给大家的。<br><br>————————————————————————<br><br>看到了这个团队一直在讨论组里接受大家的意见，我还是很欣慰的，至少他们没有撂挑子，各种敷衍，从实际行动来看，诚意还是有的。<br><br>实际上，我现在明白了一些道理。国产游戏其实要得并不是情怀支持一般的娇生惯养，而是需要玩家以一视同仁的眼光来看到它们，就如同看待国外VR游戏一样。不喜欢了，我就喷，做得好我就支持，给更多的意见。然后是国产团队脚踏实地的来接受并且认真做好游戏。如果这样的环境有，那么我希望发生在国产VR游戏身上！<br><br>好了，有什么意见和建议请留言告知，我们下一个好玩的VR游戏见。<br><br>					<div class="gradient"></div>

					　　《伊恩(Aeon)》是一款第一人称射击VR游戏。伊恩，一位来自北欧神话里掌握时间的战神，在虚拟世界中复活。玩家需要拿着两把锋利的武士刀和一副大口径的火枪，即可体验到爽快的爆裂设计。同时还能利用你的武士刀随心所欲的斩杀敌人。<br><br>　　VR上的射击游戏数不胜数，我们也是玩过了太多不同风格主题的射击游戏，所以起初对于游戏并没有什么特别的印象，因为在饱受宣传视频的“欺骗”后，我们完全不觉得酷炫的视频和图片会给游戏加分。<br><br>　　不过这种想法在体验过游戏之后就完全打消了，可以说《伊恩》算是拿出很大的诚意了。游戏操作很简单，双枪双刀的配置其实对于老玩家来说已经不是什么稀奇的配置了，相对来说，这种配置在动作射击类游戏中还算是比较“寒酸”的了，但是丝毫没有给游戏体验减分。<br><br>　　正式进入游戏，我们看到的第一个关卡的场景设计，感觉与《燥热VR》的画风有些相似，纯色的只有几何图像建立的世界，不过敌人的造型设计完全不同，《伊恩》中的敌人不是红色低面风格的小人，而是设计未来化的机器人。<br><br>　　游戏是站桩式的射击方式，但关卡场景中有瞬移的定点，所以也不算是站在同一个地方一直打到结束，再加上玩家可以自行移动身体闪避攻击，所以相比之下会比较有趣一些。游戏中瞬移的画面特效也是有精细设计过的，有种空间扭曲的感觉。<br><br>　　子弹时间算是射击游戏中一个比较特别的配置，不过现在在VR游戏中也算比较常见了，对于喜欢射击游戏的玩家来说并不陌生。<br><br>　　游戏中玩家本身扮演的就是时间战神，所以控制时间自然是必备的招式，有意思的是游戏的子弹时间完全没有CD，只要你愿意整场都用子弹时间都行。但手柄上的侧面按键在游戏中不太方便长时间操作，加上游戏子弹用光后需要到腰间直接换枪，操作上会有一定的难度。<br><br>　　游戏以时间战神伊恩为主题，自由操纵子弹时间的设计也算是非常符合题目的，让玩家体验到不同的射击乐趣，相对有限制性的子弹时间操作，这样的设计来得更加爽快。<br><br>　　除了子弹时间，游戏还有切割系统，这个玩法虽然并不算陌生，但要想做到不错的操作效果却没有那么简单。很多游戏尝试加入这个玩法，却因为呈现的效果和手感并不理想，反而成了游戏的败笔。<br><br>　　出人意料的是游戏的切割玩法手感竟然还不错，不像之前有些游戏中的固定切面，游戏完全是依照玩家砍中的位置立刻计算出横切面的展现方式，所以切割效果很有画面感。玩家们完全可以依照自己的意愿操作，想把敌人大卸八块都是随意的事儿。<br><br>　　画面效果还不赖的情况下，武士刀的力度反馈也很精确，手柄震动恰到好处，操作起来十分顺畅，几乎是手起刀落敌人就人头落地的节奏，感觉特别刺激。除此之外，游戏的射击体验也很带劲，虽然武器种类比较单一，但操作手感却是很多游戏不能比拟的。<br><br>　　游戏比较特别的体验还是子弹时间下的切割操作，玩家不仅可以砍人，还可以对敌人射击过来的子弹进行切割。不要以为这些逆天的技能就会让游戏的难度有所减少，还是有操作切换的难度，加上四面八方不断出现的敌人，只要你一个反应不过来就Game Over了。<br><br>　　游戏带给玩家一种奇妙的动作体验，将子弹时间完美得融合到动作射击的操作中，使得一成不变的射击游戏有了不一样的操作体验。<br><br>　　除了操作上让人惊艳的表现，游戏在画面建模上也丝毫没有马虎，整个场景制作都很细致，在感官上就给玩家一个很好的印象分了。不过游戏一共两个关卡，却有着两种截然不同的画风也是够分裂的。<br><br>　　游戏在第一关还是很简洁却很有设计感的几何空间，配合上机器人造型的敌人，感觉很有未来感。而第二关画风突然一转，变成了写实风格，俨然一副军事基地的场景，让人感觉有点穿越。<br><br>　　另外游戏的特效也是加分项，第一关场景中的几何建筑是可以击碎的，在敌人躲在掩体后面时，玩家可以用枪击碎掩体，碎片会漫天飞舞，配合着枪林弹雨的效果显得画面很有感觉。第二关较现实的场景中就是各种火花带闪电的效果了，金属碰撞的感觉也特别带感。<br><br>　　游戏暂时只有两个关卡比较令人遗憾，虽然有个online的模式，但是目前完全匹配不到其他玩家，所以根本无法体验。但是不得不说《伊恩》算的上是国产游戏中非常出色的射击游戏了。<br><br>　　总得来说，游戏是值得赞赏的，作为一款国产游戏我们看到设计者的用心，对比过去很多的射击类游戏，无论是国内还是国外的，《伊恩》都有着自己的优势。不过不太近乎人意的是目前游戏的关卡内容太少，体验不过瘾，不过官方称日后会不断增加内容以及新的玩法，所以还是很值得期待的一款游戏。<br><br>　　目前《伊恩(Aeon)》在steam上的售价为57元。折后的价格还算比较合适，对于目前内容较少的游戏来说88元的原价有些偏贵了，推荐喜欢动作射击游戏的玩家购买体验，也期待游戏新内容的更新。<br>
