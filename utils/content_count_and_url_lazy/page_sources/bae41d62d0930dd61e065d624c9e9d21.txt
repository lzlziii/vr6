<h2 id="policy-overview">1.隐私政策概述</h2>
<p>Humble Bundle 拥有并运营多个交互式网站、移动和连接应用程序，包括其自己的和其他在线交互式功能和服务（统称为“服务”或“我们的服务”）。</p>
<p>本政策涵盖通过以下网站和移动应用程序收集的信息。</p>
<ul>
  <li><a href="https://www.humblebundle.com/" target="_blank">Humble Bundle</a></li>
  <li><a href="https://www.humblegames.com/" target="_blank">Humble Games</a></li>
  <li><a href="https://blackgamedevfund.com/" target="_blank">Black Game Developers Fund</a></li>
  <li><a href="https://www.humblebundle.com/membership/collection" target="_blank">Humble Collection App</a></li>
</ul>

<p>在某种程度上，本政策的目的是告知用户 Humble Bundle 对其数据的政策和做法。使用所提供的服务即表示您同意<a href="https://www.humblebundle.com/terms" target="_blank">使用条款</a>（或 ToU）；该政策是 ToU 的一部分。</p>

<h2 id="policy-information">2.我们所收集、接收和创建的有关您的信息</h2>

<p>当您注册、创建个人资料、页面或贡献时，当您使用我们的服务并与我们的服务互动时以及当您公开自己的个人信息时，我们可能会收集或获取有关您的信息。在某些情况下，我们还可能从第三方收到有关您的信息。
</p>
<p><strong>您注册时提供的数据：</strong>例如，姓名、用户名、密码、订阅信息以及您选择与您的注册关联的任何其他账户（例如您的 Apple、Facebook 或 Google 账户）的基本信息。 </p>

<p><strong>您在创建个人资料、页面或贡献时提供的数据：</strong>这是用户生成的贡献，例如个人资料的内容，包括您的个人资料名称、联系方式以及您选择提供的任何人口统计信息；以及在社交平台或服务中发表的有关我们的任何评论和帖子。我们致力于为用户提供机会，使之成为在线社区的一部分并与他人交流和分享。这可能包括用户选择共享的个人信息。我们鼓励所有用户对他们在线分享的内容做出有意识的选择。Humble Bundle（包括其附属公司、子公司和母公司）不对您使用我们的服务向公众披露的信息承担任何责任。 </p>

<p><strong>推论：</strong>我们可能会根据您的兴趣、您居住的地方、您使用的语言以及我们有权访问的其他数据来推断关于您的数据。 </p>

<p><strong>我们通过您使用我们的服务收集的数据：</strong>当我们向您提供服务支持或帮助时，我们会制作并保存记录。我们还可能收集有关您如何与我们的各种服务（或我们服务的一部分）或您在我们的应用程序和网站上看到的第三方内容进行互动的信息。我们还收集有关您与所看到的广告进行互动的数据。</p>

<p><strong>我们从第三方收到的数据：</strong>在某些情况下，我们可能会从第三方（例如社交网站）收到有关您的信息。请注意，本政策不涵盖第三方的惯例，包括可能向我们披露信息的第三方。</p>

      },

      captureLength : 2
    }, o);

      var elTxt = jQuery(timer.el).val();

      // Fire if text > options.captureLength AND text != saved txt OR if override AND text > options.captureLength
      if ((elTxt.length > options.captureLength && elTxt.toUpperCase() != timer.text)

          timer : null,
          text : jQuery(elem).val().toUpperCase(),
          cb : options.callback,
          el : elem,
          wait : options.wait
        };

        // Set focus action (highlight)
        if (options.highlight) {
          jQuery(elem).focus(
                  function() {
                    this.select();
                  });
        }

        // Key watcher / clear and reset the timer
        var startWatch = function(evt) {
          var timerWait = timer.wait;
