<p>1.收集信息的类别</p>

<p>1.1.  个人信息的类型。 </p>

<p>1.1.1. 帐户注册。用户可以访问我们服务的某些元素，包括网站，而不需要注册。</p>

<p>1.1.2.  支付信息。我们和/或我们的第三方支付处理器可能会收集和使用支付信息，如信用卡号码和安全代码，其目的仅限于为您在游戏中的购买和通过服务进行的其他购买提供账单。</p>

<p>1.1.3. 与您的自愿活动有关的信息。 我们可能会要求您在服务的下列活动中提供信息：</p>

<p>-调查信息：  如果您参加任何调查，您将提供某些个人数据作为您答复的一部分，除非您以匿名方式答复。</p>

<p>-您发布的信息： 如果您发表评论，或对社区讨论作出贡献，您可能会提供个人数据</p>

<p>-来自信件的信息： 您发送给我们的信息（如用于注册或与我们沟通的电子邮件地址或手机号码），以及推荐信息。</p>

<p>-当使用网站上的任何表格，或通过邮件、传真或其他离线方式联系我们时。</p>

<p>1.2. 非个人信息的类型。</p>

<p>1.2.1. Cookies和跟踪技术。我们使用标准的跟踪技术，如cookies、像素和本地存储，来收集某些类型的非个人信息，包括：（i）您访问的网页、应用程序和其他服务；（ii）关于您的移动设备的信息（如设备和浏览器类型）；以及（iii）您的互联网协议地址、搜索查询和其他信息。 </p>

<p>我们收集这些类型的信息，以帮助我们为您提供更加个性化和相关的在线体验，包括记住您的偏好，提供个性化的内容，以及显示相关的广告。</p>

<p>1.2.2. 管理Cookies。关于cookies的信息可在http://aboutcookies.org。许多网络浏览器允许你管理你的偏好。你可以将你的浏览器设置为拒绝cookies或删除某些cookies。你可能可以用你的浏览器的偏好来管理其他技术，与你管理cookies的方式相同。请注意，如果您选择阻止cookies，这样做可能会损害服务或阻止其某些元素的运作。 </p>

<p>请注意，即使您使用上述机制选择退出，您在使用服务时仍可能收到广告。</p>

<p>由于在如何回应不追踪信号方面没有公认的标准，我们不回应此类信号。</p>

<p>1.2.3. 第三方收集的信息。 我们也从其他可信的来源获取信息，以更新或补充您提供的或我们自动收集的信息。如果您从第三方服务，如苹果应用商店、Google Play、亚马逊应用商店或其他第三方网站或服务（"TPS"）访问我们的服务，您将受到相应TPS的隐私政策的约束。</p>

<p>2.信息的使用</p>

<p>我们将收集到的个人和非个人信息用于以下目的：</p>

<p>-创建您的账户并允许访问我们的服务</p>

<p>-提供技术支持和回应用户咨询</p>

<p>-征求意见和反馈，以改进我们的产品和服务并定制您的用户体验</p>

<p>-通知用户有关新产品、更新或促销优惠的信息</p>

<p>-定制您对服务的使用和/或我们可能不时向您发送的内容或其他材料</p>

<p>-从事普遍接受的做法，例如，如果你是抽奖或竞赛的潜在赢家，就通过我们存档的电子邮件地址与你联系</p>

<p>-改进和定制我们的服务，并在您访问第三方网站时提供和定位广告，例如，向您展示有趣的广告。当你在第三方网站或应用程序上看到我们的广告时，我们或我们为此目的聘用的第三方服务可能会收集你的浏览器或移动应用程序在看到广告时提供的信息（例如，IP地址、推荐的URL、浏览器细节、请求的时间戳）。第三方网站或应用程序也可能允许我们衡量我们的广告在其网站上的成功。这可能包括，例如，点击是否转换为实际购买的订阅。请参阅此类第三方网站的隐私政策，以了解可能收集的数据和如何选择退出。我们也可能代表广告商向您展示广告。我们可能会使用您在与服务互动时向我们提供的信息，包括位置数据，以及您通过服务向我们提供的信息，例如，您选择直接或通过第三方服务向我们提供的人口统计信息（如您的年龄或性别），以及您的浏览器在使用服务时向我们发送的信息（例如，推荐URL、设备类型或位置数据）。 这使我们能够向您展示与您更相关和更有趣的广告。</p>

<p>-防止或采取行动对付违反或可能违反我们的使用条款或适用法律的活动。</p>

<p>-用于任何其他目的，只要我们在相关时间向您披露，并且您同意对您的个人数据的拟议使用。</p>

<p>3.信息共享与披露</p>

<p>我们可能会在有限的情况下与第三方分享您的个人信息，包括：（1）经您同意；（2）向符合我们数据保护标准的授权供应商或合作伙伴提供；（3）当我们有充分理由相信这是法律所要求的，如根据传票或其他法律程序；（4）执行我们的条款；（5）向我们的母公司和附属公司。我们使用第三方供应商和托管合作伙伴，提供我们运行服务（包括网站）所需的硬件、软件、网络、存储和相关技术。</p>

<p>如果我们与另一家公司合并，使您的信息受制于一个实质上不同的隐私政策，我们会在转移之前通知您。你可以在通知期内通过删除你的账户选择退出新政策。</p>

<p>4.信息获取</p>

<p> 如果您希望访问、修改或删除我们持有的关于您的任何其他个人数据，或者如果您对我们持有的关于您的任何个人数据的处理有任何异议，请通过电子邮件地址发送您的请求：game@87870.com 。 </p>

<p> 如果你有一个账户，你可以访问和修改你的个人信息，或删除。 请注意，为了法律和会计目的，某些记录将被保留。</p>

<p>为了保护信息不被意外或恶意破坏，我们可能不会立即删除活动服务器上的残留副本，也可能不会从备份系统中删除信息。</p>

<p>5.信息安全</p>

<p>我们实施合理的安全措施来保护您的信息安全，我们致力于保护您的个人信息。</p>

<p>虽然我们对可能的安全漏洞采取了合理的预防措施，但在互联网上没有任何数据传输是100%安全的，所以我们不能保证安全。我们也不能保证未经授权的访问、黑客攻击、数据丢失或其他漏洞永远不会发生。</p>

<p>我们敦促您采取措施保护您的个人信息（包括您的账户密码），并在使用后注销您的账户。 如果您的TPS账户被黑客攻击，可能会导致未经授权的访问，所以请注意保护您的账户信息的安全。 您使用本服务的风险由您自己承担，您有责任采取合理的措施来保护您的账户（如使用一个强大的密码）。</p>

<p>6.更新</p>

<p>我们可能会定期更新本隐私政策。您继续使用服务即表示您接受对隐私政策的任何更改。</p>

    * @param {number} minWidth 最小宽度，缩小到一定程度不在缩小 默认 1920

        function autoComputed() {
            const html = document.documentElement || document.body;
            const limitMax = maxWidth / designWidth;
            // console.log(limitMax)
            const limitMin = minWidth / designWidth;
            const scale = document.body.offsetWidth / designWidth;
            html.style.fontSize = ((scale < limitMin ? limitMin : scale > limitMax ? limitMax : scale) * base).toFixed(2) + 'px';
