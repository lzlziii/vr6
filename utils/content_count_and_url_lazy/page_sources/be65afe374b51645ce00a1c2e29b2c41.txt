                        </div>
                        <div id="faqs" class="col-sm-12 col-xs-12 col-md-6 col-lg-6 at_col_white">
                            <h4 class="at_h_faq">-1. How long is the gameplay?</h4>
                            <h4 class="at_h_ans">23 minutes. The objective in the game is to complete the mission in 23 minutes. That would be difficult to achieve though, since the bosses in the game require unique kill strategies and is not a simple matter of point and shoot. Fortunately though, the game has save-points after each boss fight that allow players to reload and retry infinitely.</h4>
                            <h4 class="at_h_faq">-2. What platforms will BLR80 be available for?</h4>
                            <h4 class="at_h_ans">Meta Quest/Rift and Steam</h4>
                            <h4 class="at_h_faq">-3. Does BLR80 support SteamVR?</h4>
                            <h4 class="at_h_ans">Not currently. BLR80 uses the Oculus SDK. However, <a href="https://github.com/LibreVR/Revive"><b>Revive</b></a> should help run the game on SteamVR devices.</h4>
                            <h4 class="at_h_faq">-4. Is there multiplayer or co-operative play?</h4>
                            <h4 class="at_h_ans">No, BLR80 is a single-player game.</h4>
                            <h4 class="at_h_faq">-5. Is the game in English?</h4>
                            <h4 class="at_h_ans">English is the only language supported across all UI. Some of the bot audio dialogs are in English as well as in Kannada though.</h4>
                            <h4 class="at_h_faq">-6. Are the locations in the game fictional?</h4>
                            <h4 class="at_h_ans">The city part of the game is a mishmash of fantasy and various parts of Bangalore city.</h4>
